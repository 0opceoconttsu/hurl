## NAME

hurlfmt - format Hurl files


## SYNOPSIS

**hurlfmt** [options] [FILE...]


## DESCRIPTION

**hurlfmt** formats Hurl files.



## OPTIONS



### --color {#color}

Colorize Output


### -h, --help {#help}

Usage help. This lists all current command line options with a short description.



### --html {#html}

Generate html output.



### -V, --version {#version}

Prints version information
